# Курсовой проект: база данных для фирмы грузоперевозок

Структура описывает систему управления грузоперевозками. Полностью реализованы таблицы для транспорта, водителей, грузоотправителей, грузополучателей, рейсов, грузов и оплат. SQL-скрипт формирует всю базу: объекты, связи, индексы и данные.

## Требования задания

Фирма эксплуатирует грузовые машины и выполняет перевозки. В базе фиксируются:

* характеристики транспорта (госномер, дата техосмотра, водитель, грузоподъёмность);
* грузоотправители и грузополучатели (название, адрес, расчётный счёт);
* сведения о грузе (наименование, ценность, количество);
* параметры перевозки (дата погрузки, дата доставки, статус, сторона оплаты);
* информация об оплатах.

База должна поддерживать:

* создание и изменение заказов;
* управление доставками;
* изменение данных о водителях;
* получение аналитики по перевозкам.

## Реализованные таблицы

Реализованы сущности, покрывающие все элементы задания:

* **Transport** — транспортные средства, характеристики и техобслуживание.
* **Drivers** — водители.
* **Shipper** — грузоотправители.
* **Consignee** — грузополучатели.
* **Cargo** — сведения о грузе.
* **Transportation** — рейсы, сроки, статусы, расчёт стороны оплаты.
* **Delivery** — детальная информация о доставке.
* **Payment** — расчёты по перевозкам.

Все таблицы связаны внешними ключами. Предусмотрены индексы на операционных полях.

## Запросы, которые обеспечивает структура

Система позволяет получать:

* все перевозки конкретного грузоотправителя;
* список незавершённых перевозок за указанный период;
* сумму оплат по выбранному диапазону дат.

## Содержимое репозитория

* **CURSOVIC.sql** — полный дамп базы: схема + данные.
  Может быть выполнен на пустой базе SQL Server для развёртывания всей структуры проекта.

## Развёртывание

1. Создать пустую базу данных в SQL Server.
2. Выполнить файл `CURSOVIC.sql` целиком.
3. База будет полностью восстановлена со всеми таблицами, связями и тестовыми данными.
