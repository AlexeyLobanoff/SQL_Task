# Курсовой проект: база данных для компании грузоперевозок

Проект представляет реляционную базу данных для фирмы, выполняющей грузоперевозки собственным автопарком.  
SQL-скрипт `CURSOVIC.sql` создаёт полную структуру базы (таблицы, связи, индексы) и заполняет её тестовыми данными.

---

## Описание предметной области

Фирма владеет несколькими грузовыми автомобилями и оказывает услуги по перевозке грузов. В системе учитываются:

- характеристики транспорта: государственный номер, дата прохождения техобслуживания, грузоподъёмность;
- водители, закреплённые за машинами;
- грузоотправители и грузополучатели: наименование, адрес, расчётный счёт;
- сведения о грузе: наименование, ценность, количество;
- параметры перевозки: даты погрузки и доставки, статус, сторона оплаты (отправитель или получатель);
- информация об оплате выполненных перевозок.

База данных поддерживает:

- создание и изменение заказов на перевозку;
- регистрацию и корректировку данных по доставкам;
- изменение информации о водителях;
- формирование отчётности по выполненным перевозкам и оплатам.

---

## Структура базы данных

В модели реализованы следующие сущности:

- **Transport** — транспортные средства и их характеристики (включая дату техобслуживания и грузоподъёмность);
- **Drivers** — сведения о водителях;
- **Shipper** — грузоотправители;
- **Consignee** — грузополучатели;
- **Cargo** — информация о грузе;
- **Transportation** — заказы/рейсы с основными параметрами и стороной оплаты;
- **Delivery** — данные о фактической доставке;
- **Payment** — информация об оплатах по перевозкам.

Между таблицами настроены внешние ключи, на основных полях созданы индексы для ускорения выборок.

---

## Типовые запросы

На основе структуры базы могут быть получены, в частности, следующие данные:

- список всех перевозок заданного грузоотправителя;
- перечень незавершённых перевозок за указанный период;
- сумма оплаты за перевозки в заданном диапазоне дат.

## Содержимое репозитория

- `CURSOVIC.sql` — полный дамп базы данных (схема + данные), пригодный для развёртывания на SQL Server;
- `README.md` — описание проекта и инструкции по развёртыванию.

---

## Развёртывание базы данных

1. Создать пустую базу данных в экземпляре SQL Server.
2. Открыть файл `CURSOVIC.sql` в SQL Server Management Studio или другом клиенте.
3. Выполнить скрипт целиком.
4. После выполнения в базе будут созданы все таблицы, связи, индексы и загружены тестовые данные, необходимые для демонстрации работы системы.
